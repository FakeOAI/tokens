# 成为 Tokens 赞助者

本页说明 Tokens 文档站的赞助位和广告位规则，以及如何成为赞助商。

### 位点与定价

| 类型   | 展示方式                                                       | 数量限制  | 定价         |
| ------ | -------------------------------------------------------------- | --------- | ------------ |
| 赞助位 | 【文档右侧边栏/程序左侧边栏】固定展示你的名称 / Logo，常驻曝光 | 最多 2 个 | 2000 元 / 月 |
| 广告位 | 【文档右侧边栏/程序左侧边栏】广告卡片，页面每次加载随机展示其中一条 | 多个轮播  | 1500 元 / 月 |

### 赞助位规则

- 当前只开放两个赞助位，先到先得。
- 每个赞助位会在右侧边栏「赞助位」区域常驻显示：
  - 优先显示你的 Logo（如配置），否则显示你的名称；
  - 点击卡片会跳转到你配置的网站链接。

### 广告位规则

- 支持配置多条广告，每条包含图片（可选）、标题、描述、跳转链接。
- 用户每次打开页面时，会从所有广告中随机选一条展示，你的广告将获得对应的曝光机会。
- 点击整块广告卡片会跳转到你配置的网站链接。

### 如何成为赞助商

1. 准备好以下信息：
   - 赞助类型：选择赞助位或广告位（也可以同时选择）。
   - 展示名称：你的品牌 / 产品名。
   - Logo 图片地址（可选，建议使用透明 PNG 或 SVG）。
   - 跳转链接：点击后希望打开的网站地址。
2. 通过文档中的「联系作者」页面与作者取得联系：
   - 说明「申请 Tokens 文档赞助」，并附上上述信息；
   - 商议投放时长（按月计费）和起始时间；
   - 完成支付后，我们会在约定时间内完成配置与上线。

配置完成后：

- 你的信息会出现在对应的赞助位或广告位上；
- 若在赞助期内需要更新 Logo / 链接 / 文案，可随时联系作者进行调整。
