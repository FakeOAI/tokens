# 平台计费标准

> [!WARNING]
>
> 程序采用 **月费授权** 方式进行售卖，按平台类型实行差异化定价。

## 计费规则

| 平台                         | 价格        |
| ---------------------------- | ----------- |
| **Claude 官网逆向**          | `300 元/月` |
| **Grok 官网逆向**            | `300 元/月` |
| **Sora 官网逆向**            | `500 元/月` |
| **ChatGPT 官网逆向**         | `100 元/月` |
| **Gemini 官网逆向**          | `400 元/月` |
| **Flow 官网逆向**            | `400 元/月` |
| **Gemini Business 官网逆向** | `500 元/月` |
| **Claude Code 终端逆向**     | `500 元/月` |
| **Gemini Cli 终端逆向**      | `500 元/月` |
| **Codex 终端逆向**           | `500 元/月` |
| **Antigravity 客户端逆向**   | `500 元/月` |
| **Cursor 客户端逆向**        | `500 元/月` |
| **Kiro 客户端逆向**          | `500 元/月` |
| **Gemini 官方API**           | `200 元/月` |
