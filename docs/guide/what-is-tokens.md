# Tokens 是什么?

Tokens 是一个**管理国外各个主流 AI 平台**的 token 管理平台，把各个平台的鉴权 token 加入到在我们的后台系统中，程序实现逆向以及 OpenAI、Anthropic、Gemini等平台标准API接口的格式转换，就可以无缝接入现在各大主流的对话软件、终端软件中。

## 原理是什么？

通过客户端逆向以及网页逆向的原理，进行反解析、解密以为伪造等多项技术绕过各大平台的安全检测，使用每个平台的鉴权 Token（JWT 鉴权、Session 鉴权等）进行各大平台核心对话接口的请求，然后通过程序的转换逻辑，转换成业界标准的 OpenAI、Anthropic、Gemini等平台标准API接口的格式。

## 为什么选择我们？

Tokens 本身是一个**商用项目**，由**专业的逆向工程师团队**持续维护，不提供开源版本。市面上可能会出现同类的开源逆向管理工具，但逆向项目的特殊性在于：各大平台会**不定期更新安全策略**，接口随时可能发生变化，开源社区的同类工具依赖志愿者维护，出现问题后往往**无法及时得到解决**，可能导致服务长时间不可用。如果您不想在逆向维护上耗费额外的时间和精力，希望**专注于自身业务**，选择 Tokens 是更省心的方案——我们帮您搞定逆向，您只管专心搞业务。以下是与开源社区同类方案的对比：

| 对比项目   | Tokens 商用版                            | 开源社区方案                       |
| ---------- | ---------------------------------------- | ---------------------------------- |
| 功能更新   | 持续迭代，最新功能与平台适配第一时间上线 | 依赖社区维护者，更新节奏不可控     |
| 技术支持   | **一对一专属售后**，问题快速定位与解决   | 社区自助，问题无法保证及时响应     |
| 故障处理   | 平台接口变动时**第一时间修复**并推送更新 | 需等待维护者空闲时修复，周期不可控 |
| 稳定性保障 | 专人持续维护，保障生产环境稳定运行       | 维护随时可能中断或停更，风险自担   |
| 平台覆盖   | 支持全部主流平台，新平台第一时间接入     | 覆盖有限，部分平台可能无人适配     |
| 部署支持   | 提供部署指导与配置协助                   | 需自行研究部署，遇到问题自行排查   |

## 使用场景

可以把此系统提供的**接口**集成到各大主流的对话软件或API分发系统中，比如：

- [NextChat](https://github.com/ChatGPTNextWeb/NextChat)

- [OneAPI](https://github.com/songquanpeng/one-api)

- [NewAPI](https://github.com/Calcium-Ion/new-api)
