储存配置 ​

储存配置主要用于储存各个平台返回的媒体文件，目前储存方式有 3 种，分别是：本地储存、Cloudflare R2 储存、阿里云OSS储存。

本地储存 ​

本地储存会将媒体文件储存在根目录 public 文件夹下面，可以在后台中设置文件的存活时间以及文件清理周期等。

Cloudflare R2 储存 ​

Cloudflare R2 储存会将媒体文件储存在 Cloudflare R2 中，可以在后台中设置 Cloudflare R2 配置参数。配置教程如下：

  1. 注册并登陆 https://dash.cloudflare.com，点击左边的 R2对象储存，然后点击右边管理按钮

  1. 点击 创建 Account API 令牌 按钮

  1. 选择管理员可读可写权限，然后点击创建

  1. 然后把这里的值依次填入后台设置中

  1. 在储存桶的设置中开启公共开发 URL 或自定义域名，配置好后，填入后台中即可

  1. 最终填写示例参考如下

阿里云OSS储存 ​

阿里云OSS储存会将媒体文件储存在阿里云 OSS 中，可以在后台中设置阿里云 OSS 配置参数。
